# 🧮 Calculator with GUI (Python)

## 📌 Project Description

A lightweight and user-friendly Calculator application developed using Python and Tkinter. This project provides a graphical interface to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. It demonstrates fundamental concepts of GUI development and event-driven programming.

---

## 🚀 Features

* Basic arithmetic operations (+, −, ×, ÷)
* Interactive GUI using Tkinter
* Keyboard input support
* Clear and backspace functionality
* Error handling for invalid inputs

---

## 🛠️ Technologies Used

* Python 3.x
* Tkinter (GUI Library)

---

## ▶️ How to Run

1. Install Python (3.x)
2. Run the following command:

```bash
python main.py
```
